# Character Holding Product - Quick Start Guide

## üöÄ Quick Start (5 minutes)

### 1. Access the Feature
```
Navigate to: OneClickCreatorPage (1-Click Creator)
or ImageGenerationPage (Image Generation)
```

### 2. Select Use Case
```
Click on: "Character Holding Product"
(Next to "Change Clothes")
```

### 3. Upload Images
```
Image 1: Person/character to feature (required)
Image 2: Product to hold (required)
```

### 4. Choose Product Focus
```
- full-outfit: Whole outfit being held
- top: Upper garment
- bottom: Pants/skirt
- shoes: Footwear
- accessory: Jewelry, bags, etc.
```

### 5. Run Analysis
```
Click: "Analyze"
System will analyze both images in 1-2 minutes
```

### 6. Review & Generate
```
Accept recommendations or customize
Click: "Generate Images" (2 images)
Click: "Generate Videos" (20s video)
```

## ‚ùì FAQ

**Q: What's the difference from "Change Clothes"?**
A: With Change Clothes, the character wears the product. With Character Holding Product, they hold/present it. Use this for affiliate marketing and unboxing.

**Q: Can I use any images?**
A: Yes! Image 1 should have a clear person visible. Image 2 should show the product clearly. Good lighting helps.

**Q: How long does it take?**
A: ~5-10 minutes total:
- Analysis: 2-3 minutes
- Image generation: 2-3 minutes  
- Video generation: 3-5 minutes

**Q: What resolution are the images?**
A: 1024x1024 (high quality, 8K equivalent)

**Q: Can I regenerate different poses?**
A: Yes! Run the generation again with different settings (lighting, scene, mood) for different poses.

**Q: Is the video suitable for social media?**
A: Yes! 20 seconds is perfect for Instagram Reels, TikTok, YouTube Shorts.

## üîß Troubleshooting

| Problem | Solution |
|---------|----------|
| Product not visible | Try better lighting, ensure product image is clear |
| Character looks different | Check Image 1 is correct reference, use "Keep Same" hairstyle |
| Hands look weird | Try different camera angle, ensure good hand visibility in setup |
| Colors wrong | Use neutral lighting, ensure product colors are saturated |
| Generation failed | Check internet connection, try again, contact support if persistent |
