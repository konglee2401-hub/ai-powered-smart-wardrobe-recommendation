API Calls:
├─ Frontend → Backend endpoints
│  ├─ GET /api/shorts-reels/videos/upload-status
│  ├─ POST /api/shorts-reels/videos/upload-to-drive (batch)
│  └─ POST /api/shorts-reels/videos/{id}/upload-to-drive (single)
│
└─ Response format:
   {
     "downloaded": 7,      // Total downloaded
     "uploaded": 3,        // Successfully uploaded
     "uploadFailed": 0,    // Failed uploads
     "pendingUpload": 4,   // Waiting in queue
     "withAssets": 5       // With generated assets
   }

State Management:
├─ uploadStatus    → Current upload statistics
├─ uploadLoading   → Batch upload in progress
├─ uploadingVideoId → Single video being uploaded
└─ autoRefresh     → Auto-refresh toggle state

Error Handling:
├─ Try-catch blocks for all API calls
├─ User-friendly alert messages
├─ Graceful error handling
└─ No page breaks on API failure

Performance:
├─ Efficient hook usage (useEffect, useState)
├─ Proper cleanup of intervals on unmount
├─ No memory leaks
├─ Minimal re-renders
└─ Debounced refresh intervals
